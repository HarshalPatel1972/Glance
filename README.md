# Glance

A floating screen snip tool that lets users pin any part of their screen as a draggable reference overlay — without losing their place.

## Problem Solved

When working in a browser, users often need to reference something on screen while also working somewhere else — leading to constant back-and-forth scrolling. Glance lets users snip any part of a webpage and keep it pinned on screen as a floating widget, always visible no matter how much they scroll or navigate.

## Features

- **Snip Selection:** Quickly snip any part of the screen.
- **Floating Widget:** Keeps the snip pinned as a draggable, resizable overlay.
- **Variable Opacity:** Slide to adjust the transparency of the pinned snip.
- **Save Snips:** Store up to 20 snips locally.
- **Re-inject Snips:** Use the popup menu to re-inject a previously saved snip onto the current page.

## Installation (Load Unpacked)

1. Open Chrome and navigate to `chrome://extensions`.
2. Enable **Developer Mode** (top right toggle).
3. Click **"Load unpacked"**.
4. Select the `Glance` project folder.

## Usage

- Press `Alt+S` (or `Cmd+Shift+S` on Mac) to enter Snip Mode.
- Click and drag over the desired area.
- Watch as the floating widget appears.
- Use the toolbar on the widget to adjust opacity, save, or close.
- Press `Escape` to cancel Snip Mode.
